# 喜帖填寫表單

一個美觀、現代的婚禮邀請回覆表單，支援響應式設計。

## 功能特色

- ✨ 美觀的漸層設計
- 📱 響應式設計，支援手機和桌面裝置
- ✅ 完整的表單驗證
- 🎨 流暢的動畫效果
- 💝 友善的使用者介面

## 表單欄位

### 基本資訊
- 姓名（必填）
- 聯絡電話（必填）
- 電子郵件（選填）

### 出席確認
- 是否參加（必填）
- 參加人數（如選擇參加則必填）

### 飲食偏好
- 素食/全素/清真/過敏選項
- 特殊需求備註

### 祝福留言
- 給新人的祝福訊息

## 使用方式

1. 直接在瀏覽器中開啟 `index.html` 檔案
2. 填寫表單並提交
3. 表單數據會顯示在瀏覽器控制台（F12）

## 自訂化

### 修改樣式
編輯 `styles.css` 檔案來改變顏色、字體等樣式。

### 連接後端
在 `script.js` 中的 `sendToServer()` 函數中取消註解並設定您的 API 端點。
